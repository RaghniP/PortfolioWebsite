Cryptography Project

Brief:

To develop a basic website which implements various cryptographical substitution techniques , namely

1.Caesar Cipher Technique
2.Vigenere Cipher Technique
3.One time Padding
4.Hill Cipher Technique

Execution:

This project was executed using a combination of HTML,CSS and PHP. HTML and CSS was used for front end design. PHP was used to write code for encryption and decryption in various techniques.

Challenges:

The major challenges I faced during this project were:
1. I struggled initially to link java/python code for these techniques to our original html page.
2.Hill cipher technique in decryption is perticularly challenging to write as in involves complicated matrix operations.
3. Representing these techniques in user friendly manner.

Solutions:

Though there was initial struggle in trying to find ways of integrating different languages with html, I finally came to the conclusion that php is the most effective and easiest way forward for me in the given timeline.
Hill cipher decryption code took quite some time, but eventually by consulting experts and some online libraries ,it was smoothly implemented.
Front-end development is a creative process,we learn as we go along and with more experience.
